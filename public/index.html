Learn Express.js landing page

Creating a simple REST API backend in Node.js using the Express.js framework.

Available routes

GET /movies

Return the list with only id and title of all available movies.

GET /movies/:id

Return the full data of a specific movie.

POST /movies

Create a new entry in the database.

PUT /movies/:id

Update the full data of a specific movie.

DELETE /movies/:id

Remove an entry from the database.
